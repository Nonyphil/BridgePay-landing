# BridgePay Landing Page
## Project Description
BridgePay is a responsive landing page for a fictional international money transfer service. The website showcases a modern hero section, transfer form, and feature cards using a clean blue and navy colour scheme.
## Tech Stack
-HTML5
-CSS3
-Flexbox
-Responsive Design (Media Queries)
## Live Demo
https://Nonyphil.gitHub.io/bridgepay-landing/
## How to Run Locally
Clone the repository:
'' 'bash
git clone https://github.com/Nonyphil/bridgepay-landing.git
'''
Open the project folder.
Double - click **index.html or open it in your Web browser.
No installation or dependencies are required.
