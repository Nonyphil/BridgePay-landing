# BridgePay Landing Page
## Project Description
BridgePay is a responsive landing page for a fictional international money transfer service. The website showcases a modern hero section, transfer form, and feature cards using a clean blue and navy colour scheme.
## Tech Stack
-HTML5
-CSS3
-Flexbox
-Responsive Design (Media Queries)
